# Sistema de Pedidos Multihilo - Examen 3er Parcial

## Descripción
Sistema desarrollado en Java con arquitectura MVC que gestiona pedidos mediante concurrencia y persistencia en MySQL.

## Estructura del Proyecto
- **src/**: Código fuente (Modelo, DAO, Hilos, Vista, Conexión).
- **Script_DB.sql**: Script para la creación de la base de datos y tablas.

## Tecnologías
- Java (Threads & JDBC)
- MySQL
